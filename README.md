# TokenEater

**Multi-Agent система** на бесплатных моделях через OpenRouter с автоматической коррекцией ошибок.

Экспериментальный проект, в котором несколько агентов работают вместе как команда.

### Как устроена система

Система разделена на 4 уровня:

1. **GATEKEEPER** — анализ сложности запроса и первичная маршрутизация
2. **BRAIN** — глубокое планирование и разбиение задачи на микротаск
3. **Агенты** — выполнение конкретных задач (исследование, креатив, код, критика и т.д.)
4. **Final Assembly** — обобщение, проверка качества и финальная компоновка ответа

### Особенности

- Работает полностью на бесплатных моделях OpenRouter
- Автоматическая проверка и коррекция ответов агентов
- Высокая детализаця и качество финальных ответов
- Возможность обрабатывать сложные, многоэтапные запросы

### Цель проекта

Создать систему, которая может решать достаточно сложные задачи лучше, чем одна модель, за счёт разделения труда и взаимной проверки агентов.
